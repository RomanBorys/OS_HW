Завдання 2

### Встановлення SSH-сервера

```bash
sudo apt install openssh-server -y
```

SSH-сервер було успішно встановлено в систему.

Запуск та активація сервісу

sudo systemctl enable ssh
sudo systemctl start ssh

Сервіс SSH було активовано та додано до автозавантаження системи.

Перевірка статусу сервісу

systemctl status ssh

Сервіс перебуває у стані `active (running)`, що підтверджує його коректну роботу.

Генерація SSH-ключа

ssh-keygen

Було створено SSH-ключі для автентифікації без використання пароля.

Додавання ключа на локальний сервер


ssh-copy-id roman@localhost

Публічний ключ успішно додано до авторизованих ключів користувача.

Налаштування SSH config

nano ~/.ssh/config

Конфігураційний файл:

Host myserver
    HostName localhost
    User roman
    IdentityFile ~/.ssh/id_rsa

Додано псевдонім `myserver` для швидкого SSH-підключення.

Перевірка підключення

ssh myserver

Підключення до локального SSH-сервера відбувається без запиту пароля, що підтверджує коректну роботу ключової автентифікації.

Завдання 3

Створення тестового файлу

echo "test" > test.txt

Було створено тестовий файл із базовим вмістом.

Передача файлу через SCP

scp test.txt roman@localhost:~/server_sync/

Файл успішно передано до каталогу призначення на локальному сервері.

Синхронізація через RSYNC

rsync -av ~/sync_folder/ roman@localhost:~/server_sync/

Виконано синхронізацію локальної директорії з віддаленим каталогом.

Перевірка через SFTP

sftp roman@localhost
ls server_sync
exit

Файли успішно підтверджено на сервері через SFTP-сеанс.
